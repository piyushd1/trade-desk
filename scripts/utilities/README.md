# Utility Scripts

This directory contains utility scripts for development and maintenance.

## Available Utilities

- `create_test_users.sh` - Create test users in the database
- `zerodha_reauth.sh` - Helper script for Zerodha re-authentication

## Usage

### Create Test Users
```bash
cd scripts/utilities
./create_test_users.sh
```

### Zerodha Re-authentication
```bash
cd scripts/utilities
./zerodha_reauth.sh
```

## Notes

- These scripts are for development/maintenance purposes
- They may require database access or manual intervention
- Review scripts before running in production
